RULES:

3) CRITICAL PATH RULE: Every setup_commands, check_commands, and dependency_audit.cmd entry MUST start with:

4) setup_commands: run dependency/tool install for each stack before checks.

5) runtime_requirements: set from .nvmrc, engines, packageManager, requires-python, go.mod, required_version, etc. when present. Infer defaults only when the stack is detected but version files are missing.
6) check_commands: for EACH detected stack subdirectory, include every configured dimension from the table when the repo configures it — typically 3–8 commands per stack, more in monorepos. Use confidence:
